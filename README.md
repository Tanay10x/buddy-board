# Buddy Board

Competitive leaderboard and shareable trading cards for Claude Code `/buddy` companions.

## Submit Your Buddy

```bash
npx buddy-board --username yourname
```

Optional GitHub verification:

```bash
npx buddy-board --username yourname --github yourgithub
```

## Development

### CLI
```bash
cd cli && node index.js --username test
```

### Web
```bash
cd web && npm run dev
```
